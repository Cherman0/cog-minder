Cog-Minder

Cog-Minder is a helper website for Cogmind. It has a parts searcherthat takes information from the in-game gallery CSV export and turns it into this searchable webpage, and a bots bestiary that lists all bots in the game and their stats. It should have all items and bots as of Beta 10.2, but let me know if anything seems missing.

This is an MIT licensed open source project with the source available on GitHub.
